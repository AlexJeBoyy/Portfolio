Cats vs Birds

Top down shooter where you kill birds as a cat. If you kill all the birds and get the key you can open the door, you win. First project where i worked with some more complicated coding I learned from my teacher.

Key Features

  • Fish for bullets
  • Enemy raycasting
  • Lighting effects

What I Learned

This game made me learn a lot about animations and raycasting with enemy detection. I also learned a lot about making a game with a top down perspective and how to make the player move in that perspective.

Technologies Used

Programming: C# and Unity
