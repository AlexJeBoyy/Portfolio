Horse Jumping

Horse parkour game where you can get smaller to get under obstacles. You go faster over time and can choose between standing still, walking and running. I made all the assets myself in this game, I had never done this myself but i really liked doing it.

Key Features

  • Point system
  • Ducking
  • Parkour

Technologies Used

Programming: C# and Unity

Art: Aseprite
